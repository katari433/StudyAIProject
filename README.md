# StudyAI

An AI-powered study tool that helps students prepare for their classes through auto-generated flashcards and practice problems.

---

## What It Does

- Users are enrolled in one or more **classes**
- Study material (notes, content) is stored and managed via a **REST API**
- An **AI layer** processes that material to generate:
  - Flashcards
  - Practice problems
- Content is scoped per class, so everything stays organized

---

## Tech Stack

| Layer | Details |
|---|---|
| Backend | REST API |
| AI | Practice problem & flashcard generation |
| Storage | Study material per class |
